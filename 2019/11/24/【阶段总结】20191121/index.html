JS

JS安全数

JS是双精度浮点数，64位表示数字，其中1位符号位，11位为指数位，52位为尾数位(整数部分默认为1)，因此，最大的安全数是2^53。从原理上分析其实后续还可以加很多个零，但是这些数都没有办法进行加减n的操作，可以看作是不可达的。
$$
V= （-1）^s2^{E-23}(M+1)
$$
因此0.1+0.2!=0.3。

数组Array

  • 类数组指的是可以通过索引访问元素且拥有length属性的对象。但是不具备数组push、forEach之类的方法。可以通过Array.from()将类数组转化为数组，也可以利用Array.prototype.slice.call。

  • 数组的length属性可能会比数组含有的元素量多。

  • 可以利用filter和indexof进行数组间的减法

max

math和moment都有max方法，在调用时需要注意。moment针对时间可以进行add和sub的操作。

Moment.js是轻量的JS时间库，可以用来获取时间，比较时间，为时间做相关算是操作，格式化时间等。但是原生JS针对时间的操作只有new Date()，时间戳的长度为毫秒级13位，如果不是13位还需要进行相应的处理。

JSON

需要注意json对象和json字符串的区别。

需要注意的是数据在Json中的传递是以字符串的形式传递的，因此从python中产生的大数，可以通过json进行传递，但在前端解析时会判断是否在安全数内。

  • json.parse(source, reviver) 解析JSON字符串

  • json.stringify(value, replacer, space) 将JSON对象转化为字符串，replacer是可选数组/函数，只有在replacer里的属性才会被转换；space指定缩进用的空白字符串。

    • Date 日期调用了 toJSON() 将其转换为了 string 字符串（同Date.toISOString()），因此会被当做字符串处理

  • 解析Json字符串也可以利用new function和eval函数的方法

generator函数

generator函数的关键是yield关键字和函数名前的*

将异步事件同步写法了

后一个操作依赖前一个操作的结果，将回调扁平化了。

delete

delete可以用来删除对象的属性和数组元素，一般来说返回值为true。

但是当对象被Object.freeze()后则不能进行编辑，此时delete为false

Static

类（class）通过 static 关键字定义静态方法。不能在类的实例上调用静态方法，而应该通过类本身调用。这些通常是实用程序方法，例如创建或克隆对象的功能。

CSS

overflow属性

属性值有visible，hidden，scroll，auto，除visible外都会创建一个块级上下文。字长过长可以用overflow轻松解决，此外还可以设置text-overflow，对显示做进一步处理，向用户发出不同的提示信号。clip表示直接截断；ellipsis表示…；还可以设置其他字符串。

  • 设置text-overflow：ellipsis的同时一定要设置overflow的处理方式overflow:hidden
box-shadow

利用box-shadow可以设置box的阴影，参数顺序依次为：x偏移量 | y偏移量 | 阴影模糊半径 | 阴影扩散半径|阴影颜色。此外，阴影可以堆叠，不同的阴影用逗号分隔

阴影模糊半径的值越大阴影越大越模糊。可以通过将x、y的偏移值设置为0，仅设置模糊半径来达到实现四周阴影的目的。

rgba

rgba设置颜色和透明度”rgba(x,x,x,x)”

margin

要注意margin的赋值问题

  • 1值，表示4边的margin值相同
  • 2值，表示上下(纵)使用第一个值，左右(横)使用第二个值
  • 3值，表示上使用第一个值，左右(横)使用第二个值，下使用第三个值
  • 4值，表示赋值顺序为上、右、下、左顺时针方向

Antd

table

rowkey是每一行的唯一标识，如果不设置的话默认是该行所有元素的拼接。

style

修改antd自带的格式可以通过global来实现

cascader

antd的cascader组件中，option的data一定要有的字段是label、value、isLeaf（需要特别注意的是isLeaf的字段名称一定要正确，通过isLeaf是false去触发loadData中后续级别的加载）

else
  • Ant Design error: “Form.create() getFieldDecorator is not a function报错，一般来说是没有将组件包装上高阶Form.create()()，传入一个react组件返回一个新的react组件。
  • onScrollToEnd滑动到最后时触发的回调函数（设计select、cascader组件时需要考虑）

Git

网络访问

  • 在本机开发的时候localhost可以访问项目，但是用ip地址无法访问，可能是因为使用ip地址后url是http而不是https的。此外，如果这个项目的部署地址是https，是无法通过访问http相关的请求进行资源的获取的。

    原因分析，可能是由于设置了CSP安全策略，内容安全策略用来检测并削弱某些特定攻击（XSS[限制加载内容的域]、数据包嗅探攻击[指明那种协议可以被使用]）。值得注意的是对于安全策略本来只允许https，chrome-extension、filesystem的但是为了开发方便所以对http://localhost、http://127.0.0.1增加了允许。

    当在meta中也设置了csp的内容为content=”upgrade-insecure-requests”，即表示将所有不符合规范的url都改写成符合规范的https。

  • no-referrer-when-downgrade表示从https协议降为http协议时不发送referrer给跳转网站的服务器。

  • 状态码301，302分别表示永久重定向和临时重定向。

lodash

  • Pick（）返回一个只有列入挑选key属性的对象，筛选对象的属性
  • Omit（）返回一个没有列入排除key属性的对象，
  • isNil判断value是null或者undefined
  • _sortBy（待排序array，排序函数）。[python内部的sort函数是list的成员函数list.sort()，而sorted是内建函数d=sorted(c)，sort( )函数与sorted( )函数最大的区别是， sort( )函数时对已存在的列表进行操作，调用其没有返回值；而sorted( )函数是返回一个新的list,不在原来的list上进行操作，调用其返回一个排好序的list。]

除了利用nohup…&外实现命令在开发机上持续运行外，还可以利用screen。

前端代码在开发机上可以利用安装node，随后使用npm start运行。

Document

  • 在移动端开发时window.orientation可以获取屏幕旋转方向。

  • 获取浏览器高度的方式

    • window.innerHeight：获取浏览器视觉视口高度（包括垂直滚动条）。
    • window.outerHeight：获取浏览器窗口外部的高度。表示整个浏览器窗口的高度，包括侧边栏、窗口镶边和调正窗口大小的边框。
    • window.screen.Height：获取获屏幕取理想视口高度，这个数值是固定的，设备的分辨率/设备像素比
    • window.screen.availHeight：浏览器窗口可用的高度。
    • document.documentElement.clientHeight：获取浏览器布局视口高度，包括内边距，但不包括垂直滚动条、边框和外边距。
    • document.documentElement.offsetHeight：包括内边距、滚动条、边框和外边距。
    • document.documentElement.scrollHeight：在不使用滚动条的情况下适合视口中的所有内容所需的最小宽度。测量方式与clientHeight相同：它包含元素的内边距，但不包括边框，外边距或垂直滚动条。

  • 设想这样的使用场景，当用户想通过复制url达到请求一样的情况，这时候就需要把必需的参数放到url中，然后通过读取url的值进行页面的请求和刷新

    • 这时候可以利用URL中的hash（#）
    • #代表网页中的一个位置，当url中含有“#”可以在页面中搜索#后name的位置并将其滚动到可视区域（锚点）
    • 为网页位置指定标识符，有两个方法。一是使用锚点，比如<a name=”print”></a>，二是使用id属性，比如<div id=”print” >。
    • #是用来指导浏览器动作的，对服务器端完全无用。在第一个#后面出现的任何字符，都会被浏览器解读为位置标识符。这意味着，这些字符都不会被发送到服务器端（需要注意颜色标识符，需要将其转码）
    • 每一次改变#后的部分，都会在浏览器的访问历史中增加一个记录，但不会重新加载网页，使用”后退”按钮，就可以回到上一个位置。
    • 利用window.location.hash可以读取#值，onhashchange事件当hash值改变时触发。

React

  • react的渲染流程是先子后父，将子组件拼接在父组件上。
  • react的onchange的实质是onInput，而原生的onchange失焦才会触发重新渲染。
  • react使用了事件代理。
  • 在react项目开发中有一个需求是，页面滚动到tab所在位置时，tab要固定在顶部。实现的思路其实很简单，就是判断当滚动距离scrollTop大于tab距离页面顶部距离offsetTop时，将tab的position变为fixed。

Umi

umi的effect中使用的函数只是样子比较像generator函数，实际上是针对generator函数的封装。

Mobx

  • @observable 的修饰之后，属性就变成可观测状态，他们的改变会自动被观察者获知。
  • observer 函数/装饰器可以用来将 React 组件转变成响应式组件。 它用 mobx.autorun 包装了组件的 render 函数以确保任何组件渲染中使用的数据变化时都可以强制刷新组件。 observer 是由单独的 mobx-react 包提供的。
  • 当 observer 需要组合其它装饰器或高阶组件时，请确保 observer 是最深处(第一个应用)的装饰器，否则它可能什么都不做。

微信小程序

  • 微信小程序中的 wx:if 是惰性的。如果不符合渲染条件，它不会渲染相应部分；

  • 使用 display: hidden 时，元素始终渲染，只是视图层上没有显示，用户看不见。如果你的小程序有元素显示频繁切换的需求，建议你使用 display: hidden，能够为用户提供能顺畅的使用体验。

  • 名称以 bind 开头的事件不阻止冒泡，名称以 catch 开头的事件冒泡是阻止的。如 bindTap 和 catchTab。

  • 在 WXML 中，可以使用 dataset 定义 data 中的数据，会通过事件传递。它的事件以 data- 开头，多个单词以 - 链接，如 data-a-b。需要注意的是，使用这种方式定义的变量不能有大写。它会自动转成驼峰命名，调取的时候去驼峰命名的名字。
