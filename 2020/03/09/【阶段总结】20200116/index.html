JS

Json

json的本质就是字符串

json.stringfy

将js对象转化为json字符串

如果一个被序列化的对象拥有 toJSON 方法，那么该 toJSON 方法就会覆盖该对象默认的序列化行为：不是那个对象被序列化，而是调用 toJSON 方法后的返回值会被序列化

json.parse

将json字符串转为对json对象

空对象

Null、undefined、NaN
  • null和undefined都表示的是对象
  • Null指的是空值，表示不存在的元素
  • Undefined表示的是没有定义，指的是未声明或者已声明但未赋值或者不存在的元素的属性

遍历

数组的遍历

for…of

从对象数组中提取属性值为数组，还是要遍历。

对象的遍历
  • Object.keys()得到keys组成的数组
  • for…in
  • Object.getOwnPropertyNames(obj),避免得到原型链上的属性
  • Reflect.ownKeys(obj)
    • 等同于 Object.getOwnPropertyNames(target) concat(Object.getOwnPropertySymbols(target)
    • 也会返回不可枚举的属性

排序

Array.sort(function)根据function中return的数值是大于0还是小于0决定是顺序还是逆序。

因此，可以对英文字母进行排序，反正比较的是ASCII码。

精度问题

  • toFix，使用定点表示法来格式化一个数值，只考虑小数点位，需要注意的是判断位值为5的情况
    • 如果进位的位数是偶数位则直接舍弃0.595.toFixed(2)=0.59
    • 如果是奇数位则进位后舍去
  • toPrecision
    • 处理进度是从 从左到右第一个不为0的数开始的
  • JS进行除法时直接得到小数，与C不同不分商和余数，因此需要用到Math.floor和Math.ceil进行取舍以得到整数商。

HTML

空格

  • &nbsp;
    • 半角的不断行的空白格（推荐使用）
    • space键产生的空格，用空格键产生的空格不会累加，该空格占据的宽度受字体影响明显
  • &ensp;
    • 半角的空格（布局）
  • &emsp;
    • 全角的空格（布局）

canvas

canvas是不允许跨域的，canvas在处理本地文件的时候会被认为域名是空

网络

跨域

简单请求
  • 客户端
    • 请求字段里含有Origin，
  • 服务端
    • 响应字段含有Acces-Control—Allow—Origin
      预请求
  • 客户端
    • Access-Control-Request-Method: POST
    • Access-Control-Request-Headers: X-PINGOTHER, Content-Type
  • 服务端的响应
    • Access-Control-Allow-Origin: http://foo.example
    • Access-Control-Allow-Methods: POST, GET, OPTIONS
    • Access-Control-Allow-Headers: X-PINGOTHER, Content-Type
    • Access-Control-Max-Age: 86400

服务器端的响应中未携带Access-Control-Allow—Credentials：true，则浏览器不会把响应内容返回给请求的发送者。

对于附带身份凭证的请求，服务器不得设置Access-Control-Allow-Origin 的值为“*”。

跨域并不是请求发不出去，请求能发出去，服务端能收到请求并正常返回结果，只是结果被浏览器拦截了

网络状态码

502

简单来说，502是报错类型代码 bad gate way 错误的网关。产生错误的原因是连接超时，我们向服务器发送请求，由于服务器当前链接太多，导致服务器方面无法给于正常的响应，产生此类报错。

一般来说刷新就行了

常见content-type

  • application/x-www-form-urlencoded
  • multipart/form-data
  • application/json
  • text/xml

fetch与axios的对比

概括来说

  1. axios基于XHR，fetch使用了ES新规范
  2. fetch对于跨域的实现较为简单，mode
  3. fetch只对网络请求报错，因此对于400、500的错误需要封装；默认不带cookie；不支持超时控
  4. axios支持防止CSRF；提供了并发的封装。

React将字符串转为html语句

Dva

react-redux中的connect方法将store上的getState 和 dispatch 包装成组件的props,所以可以直接通过const {dispatch}=this.props使用dispatch。

dva 提供了 connect 方法。这个 connect 就是 react-redux 的 connect 。所以const {dispatch}=this.props就是由于调用了react-redux的connect的方法

Webpack

Webpack本身只能处理JS，如果要处理其他类型的文件就需要loader进行转换，所以如果我们需要在应用中添加 css 文件，就需要使用到 css-loader 和 style-loader。

Uglify

Uglify-js 是一个js 解释器、最小化器、压缩器、美化器工具集
uglify-js的发行版本只支持ES5，如果要压缩ES6+代码需要使用uglify-es。
但是umi-js(V3)用的是uglify-js，因此如果引用的包里面有es6语法的用umi脚手架就有些问题

minimizer : ‘terserjs’ webpack的配置表示不兼容不支持es6的浏览器
设置minimizer为true则会自动压缩程序

时间

如果只有年月日，“/”连接符的时间比“-”连接符的时间多八个小时

如果字符串中还含有时分秒。

如果将”-“连接的时间字符串中添加一个T则ios可以解析，但是会出现+-8小时的问题，“/”+“T”为无效时间字符串形式。

所以还是手动转为“/”连接的比较妥帖。

gzip

gzip使用了LZ77算法与Huffman编码来压缩文件，重复度越高的文件可压缩的空间就越大。

图片不需要开启gzip压缩，如果开启的话，有可能使图片变的更大。如果你注意一些网站的 img 资源时，就会发现他们都没有开启gzip
